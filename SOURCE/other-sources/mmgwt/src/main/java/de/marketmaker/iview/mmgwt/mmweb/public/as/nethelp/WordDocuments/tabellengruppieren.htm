Tabellen gruppieren

Sind Tabellen im vwd portfolio manager gruppiert, so wird diese Gruppierung auch in der vwd advisory solution übernommen. Sie können die Gruppierung in der Darstellung aufheben:

 Wählen Sie in der Toolbar der Tabelle das Icon "Gruppierung", um eine bestehende Gruppierung aufzuheben bzw. (wieder) nach diesen Spalten zu gruppieren. Die gruppierte Darstellung erkennen Sie auch an der blauen Unterstreichung im Icon.
