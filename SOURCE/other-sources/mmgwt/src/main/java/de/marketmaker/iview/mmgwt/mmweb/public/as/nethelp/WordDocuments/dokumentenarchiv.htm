Dokumenten-Archiv

Nutzen Sie das Dokumenten-Archiv zum Archivieren Ihrer Reports. Die Reports werden mit Metadaten (Titel, Typ, Bemerkung) versehen und im Dokumenten-Archiv abgelegt.

Sie können dann in der vwd advisory solution (und auch im vwd portfolio manager) jederzeit darauf zugreifen. Haben Sie z. B. einen Report im Auswertungsbereich der vwd advisory solution geöffnet, so können Sie für das aktuelle Depotobjekt (Inhaber, Depot, Portfolio oder Konto) auf alle auf Basis dieses Reports archivierten Dokumente zugreifen.
