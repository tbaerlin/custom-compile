Aktivität anlegen

 Um eine neue Aktivität anzulegen, klicken Sie rechts in der Aktivitätsübersicht auf den Button "Neu anlegen" und wählen Sie die entsprechende Aktivität in der Auswahlliste. Die auswählbaren Aktivitäten in der Liste hängen von der Konfiguration Ihres Systems ab.

Sobald Sie eine Aktivität ausgewählt haben, können Sie mit der Erfassung beginnen.
