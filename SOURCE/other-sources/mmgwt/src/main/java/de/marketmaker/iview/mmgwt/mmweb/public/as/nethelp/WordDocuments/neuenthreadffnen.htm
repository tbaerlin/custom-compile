Neuen Thread öffnen

 Wählen Sie das Icon "Neuen Thread öffnen" rechts in der Symbolleiste, um einen neuen Thread mit der Startseite zu öffnen. Die zuvor geöffneten Threads bleiben geöffnet und können über die Auswahlliste wieder in den Vordergrund geholt werden.

 Wählen Sie das Icon "Aktuellen Thread schließen" rechts in der Symbolleiste, um den gerade offenen Thread zu schließen.

 Wählen Sie das Icon "Thread wechseln", um in der Auswahlliste den entsprechende Thread zu öffnen.
