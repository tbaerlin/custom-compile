T & S

Über den Link für Times & Sales finden Sie links im Abschnitt "Historisch" die End-of-Day-Daten rückwirkend ab aktuellem Datum.

Über den Kalender können Sie das Anfangsdatum ansteuern, ab dem die Datensätze in die Vergangenheit angezeigt werden sollen.


Durch Anklicken eines Tages im Abschnitt "Historisch" wählen Sie den Tag für die Intraday-Darstellung in der rechten Tabelle aus. Zunächst sehen Sie eine halbstündige Auflösung mit der Anzahl der enthaltenen Ticks. Per Mausklick auf den Zeitbereich erscheint eine Tabelle mit einer Komprimierung auf Minutenbasis.

Erst durch erneutes Anklicken der Intraday-Einträge erscheinen dann die einzelnen Ticks mit Volumenangabe.

 Über das Icon ʺDruckenʺ in der Menüleiste oben erstellen Sie sich einen sauberen Ausdruck.

 Über das Icon ʺKomplette Zeitreihe exportierenʺ können Sie die gesamte historische Zeitreihe des ausgewählten Wertpapiers entweder als CSV- oder Excel-Datei exportieren.
