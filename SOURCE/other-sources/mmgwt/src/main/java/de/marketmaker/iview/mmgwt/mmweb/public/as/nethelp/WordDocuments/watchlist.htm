Watchlist

Beobachten Sie in individuellen Watchlists Ihre favorisierten Wertpapiere. Dabei steht Ihnen neben der tabellarischen Darstellung auch eine Chart-Galerie der Positionen zur Verfügung.

Wählen Sie den Menüeintrag "Watchlist" im Navigator, um Ihre Watchlists zu öffnen.

In der Toolbar der Watchlists stehen Ihnen folgende Funktionen zur Verfügung:

Funktion

Icon

Beschreibung

Watchlist

Links oben sehen Sie den Namen der aktuell ausgewählten Watchlist. Haben Sie mehrere Watchlists angelegt, so wechseln Sie über das Auswahlmenü zwischen den Watchlists.

Neue Watchlist anlegen

Wählen Sie dieses Icon, um eine neue Watchlist anzulegen. Geben Sie im erscheinenden Dialogfenster "Neue Watchlist anlegen" in das Eingabefeld "Name" die gewünschte Bezeichnung ein und bestätigen Sie mit "OK".

Nun können Sie der neuen Watchlist Positionen hinzufügen.

Aktuelle Watchlist umbe-nennen

Wählen Sie dieses Icon, um eine neue Bezeichnung für die aktuell geöffnete Watchlist zu vergeben. Geben Sie dazu im erscheinenden Dialogfenster "Watchlist umbenennen" in das Eingabefeld "Name" die gewünschte neue Bezeichnung ein und bestätigen Sie die Eingabe mit "OK".

Aktuelle Watchlist löschen

Wählen Sie dieses Icon, um die aktuell geöffnete Watchlist zu löschen.

Position zu Watchlist hinzufügen

Wählen Sie dieses Icon, um ein Wertpapier in die Watchlist aufzunehmen.

Ansicht

Hier können Sie zwischen der standardmäßigen Tabellenansicht und einer Darstellung der Positionen als einzelne Charts wechseln. Wählen Sie dazu den gewünschten Eintrag in der Auswahlliste "Ansicht" aus ("Tabelle" bzw. "Charts").

Spalten
konfigurieren

Wählen Sie dieses Icon, um die Spalten der Watchlist auszuwählen und anzuordnen.

Im erscheinenden Dialogfenster "Spaltenkonfiguration" wählen Sie die Spalten Ihrer Watchlist aus und legen die Reihenfolge mithilfe der Pfeil-Icons auf der rechten Seite des Dialogfensters fest.

Bestätigen Sie die Konfiguration mit "OK".

 

Bemerkungen
mehrzeilig

Wählen Sie dieses Icon, um lange Texte in der Spalte ʺBemerkungʺ vollständig anzuzeigen bzw. wieder auf eine Zeile zu reduzieren.

 

Positionen zu Watchlists hinzufügen

Um Wertpapiere zu einer Watchlist hinzuzufügen, gehen Sie wie folgt vor:

▪       Wählen Sie in einer geöffneten Watchlist das Icon "Position zu Watchlist hinzufügen".

▪       Klicken Sie in einem Suchergebnis auf das Icon vor dem hinzuzufügenden Wert und wählen Sie im erscheinenden Kontextmenü den Befehl ZU WATCHLIST und dort die entsprechende Watchlist.

Hinzugefügte Positionen bearbeiten

▪       Für bereits hinzugefügte Positionen stehen Ihnen folgende Funktionen zur Verfügung:

Funktion

Icon

Beschreibung

Börse
ändern

Wählen Sie dieses Icon links von der Position, um eine andere Börse für die Bewertung des Wertpapiers zu wählen.

Position aus Watchlist entfernen

Wählen Sie dieses Icon, um die Position aus der Watchlist zu entfernen.

       

Bemerkung
eintragen

Wählen Sie dieses Icon am rechten Rand der Position, um einen Kommentar zum entsprechenden Wertpapier in der Watchlist hinzuzufügen.

Bemerkung
bearbeiten

Wählen Sie dieses Icon, um bereits eingetragene Bemerkung zu bearbeiten.
