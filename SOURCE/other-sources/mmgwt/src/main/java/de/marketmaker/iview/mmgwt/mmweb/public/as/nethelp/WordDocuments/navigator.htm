Navigator

Rechts neben dem Bereichsmenü befindet sich der Navigator. Er dient zur Navigation innerhalb eines Bereichs.

Beispiel: Kunden und Portfoliolisten im Navigator im Bereich "Kunden"

Haben Sie ein einzelnes Objekt ausgewählt, so finden Sie im oberen Bereich des Navigators einen Steckbrief und weitere Navigationselemente. Lesen Sie dazu den Abschnitt Objektfenster.
