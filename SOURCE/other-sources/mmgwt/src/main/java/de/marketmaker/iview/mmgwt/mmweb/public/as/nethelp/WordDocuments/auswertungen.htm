Auswertungen

Unter dem Menüpunkt "Auswertungen" im Navigator stehen Ihnen die im vwd portfolio manager konfigurierten Auswertungen für den Inhaber zur Verfügung. Das können Tabellen, Reports, Reportmappen oder auch Charts sein. Auch die Einteilung dieses Menüs im Navigator orientiert sich an der Workspace-Konfiguration für Inhaber im vwd portfolio manager.
