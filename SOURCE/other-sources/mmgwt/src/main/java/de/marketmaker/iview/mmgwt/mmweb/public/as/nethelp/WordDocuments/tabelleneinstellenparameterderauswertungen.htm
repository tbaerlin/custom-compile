Tabellen einstellen (Parameter der Auswertungen)

Viele tabellarische Auswertungen im Kundenbereich haben verschiedene Parameter, die Sie einstellen können. Gehen Sie wie folgt vor, um diese – vielen sicher auch aus dem vwd portfolio manager bekannten – Einstellungen vorzunehmen:

1.    Wählen Sie in der Toolbar der Tabelle den Button "Einstellungen".

2.   Legen Sie die verschiedenen Parameter fest.

 Um einen einzelnen Parameter auf den Ausgangszustand zurückzusetzen, wählen Sie das Icon ʺZurücksetzenʺ.

Um alle Parameter zurückzusetzen, wählen Sie den Button ʺZurücksetzenʺ.

3.   Wählen Sie abschießend den Button "Übernehmen".

Die Tabelle wird mit den gewählten Einstellungen neu ausgewertet.

Detaillierte Beschreibungen der jeweiligen Parameter der Auswertungen finden Sie in der Dokumentation zum vwd portfolio manager.
