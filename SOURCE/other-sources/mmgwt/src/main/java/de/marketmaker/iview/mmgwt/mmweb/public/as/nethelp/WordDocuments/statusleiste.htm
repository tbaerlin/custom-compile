Statusleiste

Unten befindet sich die Statusleiste mit aktuellen Statusinformationen zum Programm. So können Sie Informationen zu Benutzer, Programmversion und allen freigeschalteten Modulen direkt aus der Statuszeile aufrufen. Klicken Sie dazu links auf die Versionsnummer Ihrer vwd advisory solution.

Die entsprechenden Informationen werden dann in einem Hinweisfenster eingeblendet. Alternativ können Sie diese Informationen auch im Bereich ʺHilfeʺ unter ʺInfoʺ einsehen.

Rechts in der Statusleiste können Sie die Anzeige durch einen Klick um 50% verkleinern bzw. vergrößern. Klicken Sie dazu auf die Zoomanzeige.

Zudem wird ggf. die Anzahl vorhandener Meldungen – z. B. über Auswertungen, die im Hintergrund laufen bzw. gelaufen sind – angezeigt. Klicken Sie auf die Meldungen, um Details zu erfahren. Lesen Sie zu Meldungen ggf. den Abschnitt Auswertungen im Hintergrund.
