Live-Suche

Bei der Eingabe des Suchbegriffs unterstützt Sie die integrierte Live-Suche. Anhand der eingegebenen Zeichen erscheinen diese Vorschläge in einer Popup-Liste. Dort können Sie per Mausklick oder über die Pfeiltasten der Tastatur den passenden Vorschlag auswählen und so direkt das Objektfenster des ausgewählten Objekts öffnen.
