Tabellen

 Die tabellarischen Auswertungen für Inhaber – aber auch die anderen Tabellen im Bereich "Kunden" (Vermögensübersicht, Assetklassenanalyse…) können Sie mithilfe der Funktionen in der Toolbar über den Spaltenüberschriften bearbeiten.

Diese Änderungen sind nicht dauerhaft. Beim nächsten Anmelden an der vwd advisory solution werden die Tabellen wieder in ihrer Standarddarstellung aus dem vwd portfolio manager angezeigt.

Die Spalten entsprechen den Spalten im vwd portfolio manager, auch ggf. vorhandene Daten in Fußzeilen sind verfügbar.
