Fokusliste

Die Fokusliste stellt eine Liste von Aktien und aktienähnlichen Wertpapieren dar, die vom Research der Bank beobachtet werden. Die Fokusliste wird dabei zentral im vwd portfolio manager verwaltet.

Wenn Sie die Fokusliste aus der Anlageplanungsliste innerhalb einer Aktivität speichern, enthält die Liste eine der folgenden Empfehlungen:

• Kauf

• Verkauf

• Halten

• <under review>

Die Empfehlungen der Fokusliste sind als für den Berater verbindlich konzipiert.
