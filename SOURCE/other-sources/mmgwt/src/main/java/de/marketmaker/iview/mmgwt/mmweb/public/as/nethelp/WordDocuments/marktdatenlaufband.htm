Marktdaten-Laufband

Oben im Arbeitsfenster befindet sich ein Marktdaten-Laufband. Dieser Ticker liefert Ihnen aktuelle Daten zu den wichtigsten Indizes und Kennzahlen.

Zeigen Sie mit der Maus auf einen Eintrag im Laufband, um Informationen zum Kurszeitpunkt und zur prozentualen Veränderung einzublenden. Ein Mausklick auf den Index öffnen Sie das jeweilige Marktdatenfenster.
