Pflichtfelder

 Pflichtfelder sind mit einem Sternchen markiert. Wenn ein Pflichtfeld nicht ausgefüllt ist, wird es bei einem Klick auf "Weiter" mit einem entsprechenden Hinweis hervorgehoben.

Zusätzlich wird im Navigationsbereich ein orange eingekreister Stern links vom betroffenen Schritt angezeigt. Wenn Sie mit der Maus darauf zeigen, erscheint ebenfalls der Hinweis.
