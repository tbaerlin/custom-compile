Kennzahlen

Über diesen Link erhalten Sie alle von vwd errechneten Kennzahlen. Neben den eigentlichen Wertpapierkennzahlen sehen Sie auch Angaben zu den Dividendenzahlungen.

 Über den Kalender können Sie ein Vergleichsdatum wählen, um sich die Kennzahlen für einen bestimmten Tag ausrechnen zu lassen. Dabei wird eine zusätzliche Spalte mit dem Vergleichsdatum rechts nach der Spalte "Allzeit" eingefügt.

 Über den Button ʺZurücksetzenʺ setzen Sie die Spalte wieder auf ʺAllzeitʺ.
