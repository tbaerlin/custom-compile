Musterportfolio

In Ihren Musterportfolios pflegen Sie Ihre virtuellen Portfolios. Sie können maximal 20 Portfolios mit jeweils 100 Einzeltiteln anlegen.

In Muster-Portfolios stehen Ihnen folgende Funktionen zur Verfügung:

Links oben steht der Name des aktuell ausgewählten Portfolios. Haben Sie mehrere Muster-Portfolios angelegt, so wechseln Sie über das Auswahlmenü zwischen den Muster-Portfolios.

Spalten

konfigurieren

Wählen Sie dieses Icon, um die Spalten des Muster-Portfolios auszuwählen und anzuordnen.

Im erscheinenden Dialogfenster "Spaltenkonfiguration" finden Sie im Abschnitt "Verfügbare Spalten" die auswählbaren, im Abschnitt "Ausgewählte Spalten" die anzuzeigenden Spalten. Wählen Sie die Spalten Ihres Muster-Portfolios mit den Pfeil-Icons in der Mitte des Dialogs aus. Legen Sie die Reihenfolge der Spalten mithilfe der Pfeil-Icons auf der rechten Seite des Dialogfensters fest.

Möchten Sie den ursprünglichen vwd Standard wiederherstellen, wählen Sie unten links den Button "Standard". Um den Konfigurationsvorgang abzubrechen, wählen Sie "Abbrechen". Zum Bestätigen der Konfiguration wählen Sie "OK".

Positionen zu Musterportfolio hinzufügen

Um Wertpapiere zu einem Musterportfolio hinzuzufügen, gehen Sie wie folgt vor:

1.    Wählen Sie in einer geöffneten Watchlist das Icon "Position zu Musterportfolio hinzufügen".

2.   Geben Sie im erscheinenden Suchfenster den entsprechenden Suchbegriff ein, z. B. den Namen oder die WKN. Wählen Sie im Suchergebnis den gewünschten Treffer aus und bestätigen Sie mit ʺOKʺ.

3.   Geben Sie ein, wie viele Stück des Wertpapiers Sie zum Musterportfolio hinzufügen möchten. Ändern Sie bei Bedarf auch das Datum, die Börse, den Kaufkurs und die Gebühren für die Order.

4.   Bestätigen Sie die Eingaben mit dem Button ʺKauforder ausführenʺ. Die Position wird Ihrem Portfolio hinzugefügt.

Positionen aus Musterportfolio entfernen

Um Wertpapiere aus einem Musterportfolio zu entfernen, gehen Sie wie folgt vor:

1.    Wählen Sie das Icon "Verkaufsorder ausführen" links neben dem entsprechenden Wertpapier.

2.   Geben Sie ein, wie viele Stück des Wertpapiers Sie aus dem Musterportfolio entfernen möchten. Ändern Sie bei Bedarf auch das Datum, die Börse, den Kaufkurs und die Gebühren für die Order.

3.   Bestätigen Sie die Eingaben mit dem Button Verkaufsorder ausführenʺ. Die Position wird Ihrem Portfolio hinzugefügt.

Bemerkungen hinzufügen und bearbeiten

 Um eine Bemerkung zu einer Position in der Watchlist hinzuzufügen, wählen Sie das Icon ʺBemerkung eintragenʺ am rechten Ende der Entsprechenden Position. Tragen Sie im erscheinenden Fenster ʺBemerkungʺ Ihren Text ein und bestätigen Sie mit ʺOKʺ.

▪        Um eine Bemerkung zu bearbeiten und ggf. zu löschen, wählen Sie das Icon ʺBemerkung bearbeitenʺ. Ändern oder löschen Sie dann den Text in dem erscheinenden Fenster.
