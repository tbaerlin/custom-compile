Globale Dashboards löschen

Angelegte globale Dashboards können Sie auch wieder löschen. Gehen Sie dazu wie folgt vor:

1.   Wechseln Sie über das Icon "Dashboards" im Bereichsmenü in den Bereich "Dashboards":

2.   Wählen Sie das zu bearbeitende Dashboard in der Liste aus.

3.    Wählen Sie das Icon "Ausgewähltes Dashboard löschen" in der Toolbar.

Das Dashboard wird aus der Liste der Dashboards gelöscht.
