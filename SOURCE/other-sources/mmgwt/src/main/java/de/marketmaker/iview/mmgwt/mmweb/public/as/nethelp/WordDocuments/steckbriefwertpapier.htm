Steckbrief Wertpapier

Der Steckbrief Wertpapier bei integriertem "vwd market manager financials [web]" entspricht grundsätzlich dem Steckbrief Wertpapier ohne diese zusätzliche Marktdatenversorgung.

 Zusätzlich stehen Ihnen über das Pfeil-Icon in der linken oberen Ecke aber weitere Funktionen zur Verfügung. Ein Mausklick auf das Symbol öffnet ein Menü, über das Sie direkt das Wertpapierporträt oder das Chartcenter öffnen.

  Auch können Sie das Wertpapier über dieses Menü direkt zu einer Watchlist oder einem Muster-Portfolio hinzufügen.
