Verknüpfte Objekte

Unter dem Menüpunkt "Verknüpft" finden Sie Links zu allen Objekten, die mit dem aktuell ausgewählten Portfolio verknüpft sind. Das können für Portfolios die folgenden Objekte sein:

▪        Inhaber des Portfolios

▪        Depots

▪        Konten

Per Mausklick auf diese verknüpften Objekte öffnen Sie deren Objektfenster.
