Kontextlisten

 Über die Kontextliste wählen Sie direkt ein anderes Objekt aus. Haben Sie z. B. ein Wertpapier aus einer Index-Liste ausgewählt, so hält die Kontextliste alle anderen Wertpapiere des Index zur schnellen Auswahl bereit.

 Über die Listennavigation tauschen Sie den ausgewählten Inhaber gegen den nächsten bzw. vorigen in der Liste aus.
