Passwort ändern

Wählen Sie diesen Link im Bereich "Verwaltung" im Navigator, um Ihr Passwort zum Login zur vwd advisory solution und zum vwd portfolio manager zu ändern. Geben Sie dazu im erscheinenden Dialogfenster "Passwort ändern" das alte Passwort sowie das neue Passwort (zweimal) ein und bestätigen Sie das Dialogfenster mit dem Button "OK".

Geben Sie zukünftig bei der Anmeldung das neue Passwort ein.
