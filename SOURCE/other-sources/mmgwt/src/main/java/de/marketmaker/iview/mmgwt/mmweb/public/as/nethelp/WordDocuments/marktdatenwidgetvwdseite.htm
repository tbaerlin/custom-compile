Marktdatenwidget "vwd-Seite"  

Das Marktdatenwidget "vwd-Seite" zeigt die bekannten vwd-Seiten an. Wie gewohnt können Sie in das Eingabefeld "Seite" die gewünschte Seitennummer eingeben (z. B. "5009" für "DAX-XETRA") und mit <RETURN> bestätigen, um zu dieser Seite zu wechseln.

 Im Bearbeitungsmodus haben Sie über das Icon "Konfiguration" im Marktdatenwidget "vwd-Seite" die folgende Einstellungsmöglichkeit:

Element

Beschreibung

Links im selben Fenster öffnen

Standardmäßig ist dieses Kontrollkästchen deaktiviert. Über die verlinkten Seitennummern öffnen Sie die gewählte vwd-Seite in einem neuen Fenster außerhalb des Dashboards.

Aktivieren Sie das Kontrollkästchen, dann wird über die verlinkten Seitennummern der Inhalt des Widgets durch die neue vwd-Seite ersetzt.

Sie können die Nachrichten auf eine bestimmte Kategorie begrenzen. Wählen Sie in dieser Auswahlliste die gewünschte Kategorie für die Anzeige aus.

Davon unabhängig wechseln über die Links der Wertpapiernamen auf den vwd-Seiten direkt aus dem Dashboard in das entsprechende Wertpapierporträt.

Bestätigen Sie die Konfiguration mit dem Button "Übernehmen".
