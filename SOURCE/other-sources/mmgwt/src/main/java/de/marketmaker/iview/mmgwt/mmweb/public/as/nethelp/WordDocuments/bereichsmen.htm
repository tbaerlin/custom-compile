Bereichsmenü

Auf der linken Seite befindet sich das Bereichsmenü der vwd advisory solution. Ein Mausklick auf den gewünschten Bereich öffnet die jeweilige Übersichtsseite.

Im Einzelnen wählen Sie hier folgende Bereiche:

Bereich

Dashboards

Im Bereich "Dashboards" stehen Ihnen die in der Regel individualisierbaren Startseiten zur Verfügung.

 

Kunden

Im Bereich "Kunden" stehen Ihnen die wichtigsten Funktionen für Kunden, Portfolios und Interessenten zur Verfügung. Basis hierfür ist das umfassende Portfoliomanagement-System des vwd portfolio manager.

Im Navigator finden Sie die Bereiche "Kundenlisten", "Portfoliolisten" und "Interessentenlisten". Zudem ist es hier möglich, einen Interessenten anzulegen.

Im Auswertungsbereich hilft Ihnen der aus dem vwd portfolio manager bekannte Explorer bei der weiteren Auswahl der Kunden bzw. Portfolios.

 

Märkte

Im Bereich "Märkte" stehen Ihnen die Daten aus dem "vwd market manager financials [web]" in zahlreichen Features zur Verfügung.

Im Navigator finden Sie ausgewählte Übersichtseiten als Ausgangspunkt für weitergehende Analysen.

Nachrichten

Im Bereich "Nachrichten" finden Sie neben der Nachrichtenübersicht auch die Nachrichtensuche.

Suchergebnis

Wählen Sie diesen Bereich, um zum letzten Suchergebnis zurückzukehren.

 Ist das Produkt "vwd market manager financials [web]" verknüpft, so werden die Suchergebnisse nach Depotobjekten und Marktdaten getrennt aufgelistet.

 

Verwaltung

Wählen Sie diesen Befehl, um diverse Einstellungen zu konfigurieren.

Beachten Sie, dass Sie sich nach einer Änderung Ihres Passworts sowohl an der vwd advisory solution als auch am vwd portfolio manager mit diesem neuen Passwort anmelden müssen.
