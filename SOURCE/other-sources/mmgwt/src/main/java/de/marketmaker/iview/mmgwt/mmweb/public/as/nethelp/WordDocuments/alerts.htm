Alerts

Alerts werden beim Öffnen eines Inhaber- oder Portfolio-Dashboards bzw. des Objektfensters eines Kontos oder Depots ausgewertet.

 Sind Alerts vorhanden, erscheint im Navigator ein Alert-Icon. Zeigen Sie mit der Maus auf das Icon, so erscheinen die generierten Alerts des Depotobjekts sowie die Alerts der mit diesem Objekt verknüpften Objekte (abgegrenzt durch eine Linie)

Klicken Sie auf das Alert-Icon, so wird im Auswertungsbereich die Auswertung "Überwachung Alerts" geöffnet.

Alle Informationen zur Definition der Alerts finden Sie in der Online-Hilfe des vwd portfolio manager.

Hinweis:
In Kundensitzungen werden Alerts nicht ausgewertet, dementsprechend ist auch das Alert-Icon immer ausgeblendet.
