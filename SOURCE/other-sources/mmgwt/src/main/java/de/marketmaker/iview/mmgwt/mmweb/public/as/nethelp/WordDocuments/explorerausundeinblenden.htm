Explorer aus- und einblenden

Sie können den Explorer im Auswertungsbereich aus- und auch wieder einblenden, indem Sie auf das Pfeilsymbol klicken.

Symbol

Beschreibung

Klicken Sie im Explorer auf den Pfeil, um den Explorer zu minimieren.

Klicken Sie bei zugeklapptem Explorer auf den Pfeil links oben im Auswertungsbereich, um den Explorer einzublenden.
