Reports archivieren

Im Auswertungsbereich geöffnete Reports können Sie (mit der entsprechenden Berechtigung) im integrierten Dokumenten-Archiv speichern. Gehen Sie dazu wie im Abschnitt Dokumente archivieren beschrieben vor.

Die Reports werden mit Metadaten (Titel, Typ, Bemerkung) versehen und im Dokumenten-Archiv abgelegt. Sie können dann in der vwd advisory solution (und auch im vwd portfolio manager) jederzeit darauf zugreifen.
