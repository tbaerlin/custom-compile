Reports einstellen (Parameter der Auswertungen)

Reports können verschiedene einstellbare Parameter enthalten. Gehen Sie wie folgt vor, um diese Einstellungen vorzunehmen:

1.    Wählen Sie in der Toolbar des Reports den Button "Einstellungen".

2.   Legen Sie die verschiedenen Parameter fest.

3.   Wählen Sie abschießend den Button "Übernehmen".

Der Report wird mit den gewählten Einstellungen neu ausgewertet.
