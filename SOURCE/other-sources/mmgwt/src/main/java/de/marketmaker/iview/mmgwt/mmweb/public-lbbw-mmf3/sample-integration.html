Dies ist die umgebende Seite.
Um im mm[web] ein Symbol zu öffnen wird ein Link notiert, der folgende Bedingungen erfüllt:
  • Die Zielseite (href) enthält javascript code mit zwei Statements
  • Das erste Statement (window.open()) öffnet ein neues Fenster, das sich nach Injektion des Symbols im mm[web] selbst wieder schließt.
  • Das zweite Statement (void(0)) verhindert, dass sich der Seiteninhalt der aufrufenden Seite ändert
Beispiel:
<a href="javascript:window.open('http://mmfweb.vwd.com/lbbw-mmf3/open-symbol.html#DE0007100000', '_blank', ''); void(0);">Daimler in mm[web] öffnen</a>
So sieht das dann aus: Daimler in mm[web] öffnen
