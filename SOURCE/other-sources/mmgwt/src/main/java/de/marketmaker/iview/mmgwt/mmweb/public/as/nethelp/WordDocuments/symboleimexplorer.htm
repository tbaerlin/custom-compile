Symbole im Explorer

Im Explorer sehen Sie folgende Symbole:

Element

Symbol

Beschreibung

Ordner

Ordner haben kein eigenes Symbol, die Pfeilsymbole davor wechseln zwischen "Zugeklappt" und "Aufgeklappt".

Die Bezeichnungen der Ordner entsprechen den Ordnerbezeichnung im vwd portfolio manager. Darunter finden Sie die enthaltenen Gruppen und Depotfilter sowie ggf. weitere Ordner.

Gruppe

Die Gruppen entsprechen den Gruppen im vwd portfolio manager und können Inhaber oder Portfolios und untergeordnete Gruppen und Depotfilter mit Inhabern oder Portfolios enthalten.

Depotfilter

Die Depotfilter entsprechen den Depotfiltern im vwd portfolio manager und können Inhaber, Interessenten, Portfolios und auch Sub-Depotfilter enthalten.

Tipp:
Eine Anleitung zum Anlegen von Sub-Filtern im vwd portfolio manager finden Sie in der Online-Hilfe des vwd portfolio manager.
