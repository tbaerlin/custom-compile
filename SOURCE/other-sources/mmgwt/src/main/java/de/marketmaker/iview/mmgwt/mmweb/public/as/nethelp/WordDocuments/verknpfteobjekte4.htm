Verknüpfte Objekte

 Unter dem Menüpunkt "Verknüpft" finden Sie die mit dem Interessenten verknüpften Personen.

Klicken Sie auf die verknüpfte Person, um es in dessen Objektfenster zu öffnen. Lesen Sie dazu den Abschnitt Objektfenster "Person".

Tipp:
Nutzen Sie die "Zurück-Funktion", um von dem so geöffneten Objektfenster einer Person wieder zurück zum Objektfenster "Interessent" zu springen.
