Dashboards und Kundensitzungen

In einer aktiven Kundensitzung ist das globale Dashboard nicht zu sehen. Existieren ans Depotobjekt gebundene Dashboards (Inhaber- und/oder Portfolio-Dashboards), werden diese nur dann angezeigt, wenn die zugehörige Vorlage im vwd portfolio manager als "Für Kundensitzungen geeignet" gekennzeichnet ist. Lesen Sie dazu ggf. den Abschnitt Vorlagen bearbeiten in der Online-Hilfe des vwd portfolio manager.
