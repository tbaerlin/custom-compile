Objektfenster "Inhaber"

Wenn Sie einen Inhaber z. B. aus einer Kundenliste auswählen, öffnet sich dessen Objektfenster. Hier sehen Sie oben im Navigator den Steckbrief des Inhabers. Darunter finden Sie die Stammdaten und verknüpften Objekte sowie die Auswertungen analog zur Worksheet-Einteilung im vwd portfolio manager gemäß der zugeordneten Workspace-Profile (Web).
