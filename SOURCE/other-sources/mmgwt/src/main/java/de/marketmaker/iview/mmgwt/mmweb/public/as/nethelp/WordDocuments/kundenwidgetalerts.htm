Kundenwidget "Alerts"

Das Kundenwidget "Alerts" liefert Ihnen den Überblick über die zuletzt ausgelösten Alerts für den Inhaber. Standardmäßig werden die 10 aktuellsten Alerts in der Tabelle mit den Spalten "Datum"; "Anlass", " Name" und "Meldung" dargestellt.

 Im Bearbeitungsmodus haben Sie über das Icon "Konfiguration" im Kundenwidget "Alerts" folgende Einstellungsmöglichkeiten:

Element

Beschreibung

Alert-Kategorie

Falls Sie nur Alerts einer bestimmten Alert-Kategorie sehen möchten, dann wählen Sie diese hier aus.

Alert-Kategorien legen Sie bei der Konfiguration der Alert-Definitionen im vwd portfolio manager fest.

Unterobjekte einschließen

Aktivieren Sie dieses Kontrollkästchen, wenn auch Alerts auf allen untergeordneten Depot-Objekten angezeigt werden sollen. Dann werden z. B. neben den Alerts auf der Gruppe selbst auch ausgelöste Alerts auf enthaltenen Portfolios angezeigt.

Anzahl Alerts

Standardmäßig werden die 10 aktuellsten Alerts aufgelistet. Geben Sie in dieses Eingabefeld ggf. eine andere Zahl ein.

Bestätigen Sie die Konfiguration mit dem Button "Übernehmen".
