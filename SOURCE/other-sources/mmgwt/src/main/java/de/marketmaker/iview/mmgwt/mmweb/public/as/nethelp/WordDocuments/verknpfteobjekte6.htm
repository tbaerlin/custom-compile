Verknüpfte Objekte

Unter dem Menüpunkt "Verknüpft" finden Sie Links zu allen Objekten, die mit dem aktuell ausgewählten Konto verknüpft sind. Das können für Portfolios die folgenden Objekte sein:

▪        Inhaber des Kontos

▪        Portfolio

Klicken Sie auf das verknüpfte Objekt, um es in dessen Objektfenster zu öffnen.
