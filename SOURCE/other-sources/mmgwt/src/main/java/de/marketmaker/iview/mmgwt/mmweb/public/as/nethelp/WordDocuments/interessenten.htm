Interessenten

Nutzen Sie Interessenten, um potentielle Kunden ggf. mit Adressdaten und weiteren Angaben anzulegen. Mit Interessenten können Sie dann sofort Aktivitäten im Zusammenhang mit dem Beratungsprozess durchführen.

Während Inhaber nur im vwd portfolio manager angelegt werden können, lassen sich Interessenten auch direkt in der vwd advisory solution erfassen.

Hinweis: 
Im Gegensatz zu Inhabern können Sie Interessenten keine Portfolios, Konten und Depots, sondern nur Personen zuweisen.
