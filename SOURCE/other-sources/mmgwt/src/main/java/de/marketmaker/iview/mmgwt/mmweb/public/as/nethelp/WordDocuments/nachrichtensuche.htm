Nachrichtensuche

Wählen Sie im Bereich "Nachrichten" im Navigator den Link "Nachrichtensuche", um nach bestimmten News zu suchen.

Im Abschnitt "Basisinfo" können Sie zunächst einen "Suchbegriff" eingeben und die Suche nach "Agentur" und "Zeitraum" einschränken. Über die Parameter unterhalb des Kontrollkästchens "Rubrik" können Sie die Suche noch weiter eingrenzen, z. B. nach "Unternehmen", "Branchen" oder "Länder & Regionen".

Um Suchbegriffe einzugeben aktivieren Sie das Kontrollkästchen "Basisinfo" und dort das Kontrollkästchen "Suchbegriff". Die Nachrichtensuche ist implizit eine "UND"-verknüpfende Suche. Um einen Suchbegriff mit "ODER" zu verknüpfen, geben Sie "OR" zwischen zwei Suchbegriffen ein. Die Groß-/Kleinschreibung spielt hierbei keine Rolle.

Beispiel: Eingabe "Volkswagen or DAX"

Diese Eingabe findet alle Nachrichten, in denen das Wort "Volkswagen" oder das Wort "Dax" vorkommt.

Möchten Sie nach einem vollständigen Begriff suchen, so können Sie den Begriff in einfache Anführungszeichen einschließen, z. B. 'DAX 30'.

Konfiguration der Suchparameter

 Über das Icon ʺSuchparameter konfigurierenʺ öffnen Sie einen Dialog, in dem Sie auswählen können, welche Suchparameter über die Kontrollkästchen für die Suche bereitstehen sollen. Auswählbar sind hierbei:

• Suchbegriff

• Agentur

• Zeitraum

• Wertpapier

 Um für die Suche gezielt ein Wertpapier auszuwählen, wählen Sie den Button ʺAuswahlʺ und suchen Sie das gewünschte Wertpapier im entsprechenden Dialog.
