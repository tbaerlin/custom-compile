Chart

Über den Link "Chart" lassen sich Wertpapiere mithilfe der technischen Analyse auswerten. Dabei können Sie vier verschiedene Charttypen einstellen, zusätzliche Vergleichswerte einblenden und Indikatoren anzeigen.

Sie können in dem Chart folgende Einstellungen vornehmen:

Vergleichswerte

 In dem Eingabekasten ganz links können Sie Vergleichswerte hinzufügen, indem Sie auf das Pluszeichen klicken und über die Suche einen Titel an einem bestimmten Börsenplatz auswählen.

 Über das Minuszeichen können Sie Vergleichswerte jederzeit wieder löschen.

 Vorhandene Vergleichswerte können Sie mithilfe der Kontrollkästchen aktivieren oder deaktivieren.

Möchten Sie als Vergleichswert einen Index hinzufügen, dann bietet sich die Auswahlliste "Benchmark" an. Dort finden Sie alle wichtigen Indizes zur Auswahl.

Charttyp

Mit der Auswahlliste "Charttyp" lässt sich die Darstellung der Zeitreihe ändern. Neben dem Linienchart stehen hier auch die Darstellungsarten Flächenchart, Balkenchart, Open/High/Low/Close sowie Candlesticks zur Verfügung.

Zeitraum

Standardmäßig erhalten Sie den 1-Jahres-Chart. Über die Auswahlliste können Sie diesen Zeitraum auch ändern.

 Alternativ können Sie das Datum über die Kalenderfunktion einstellen. Wählen Sie dazu zunächst in der Auswahlliste "Zeitraum" den Eintrag "Von-Bis".

Benchmark

Mit der Auswahlliste ʺBenchmarkʺ können Sie die wichtigsten Indizes direkt als Benchmark im Chart einblenden.

Indikator

Mit dieser Auswahlliste können Sie Indikatoren zum Chart hinzufügen. Zur Auswahl stehen die Indikatoren Momentum, Rate-of-Change, Relative Strength, MACD, Slow Stochastik, Fast Stochastik, Variabler MA und Bollinger Bands.

Währung                  

Mit dieser Auswahlliste können Sie Titel unterschiedlicher Währung miteinander vergleichen.

Gleitender Durchschnitt

Mit den Kontrollkästchen "38", "100" und "200" Tage können Sie den Gleitenden Durchschnitt über die entsprechende Anzahl von Tagen einzeichnen.

Intraday mit

Im Intraday-Bereich können Sie mithilfe der Kontrollkästchen entscheiden, ob Geld- oder Briefwerte angezeigt werden sollen.

Chart groß

Mit diesem Kontrollkästchen können Sie eine kleine oder große Chartdarstellung einstellen. Das Kontrollkästchen ist standardmäßig deaktiviert, die Darstellung also klein.

Chart Druck farbig

Mit diesem Kontrollkästchen bestimmen Sie, ob der Chart farbig oder schwarz-weiß gedruckt wird.

Volumen

Mit diesem Kontrollkästchen aktivieren Sie die Anzeige des Volumens unterhalb des Hauptcharts.

Log. Skalierung

Mit diesem Kontrollkästchen wechseln Sie von absoluter auf logarithmische Darstellung.

Absolutwerte

Mit diesem Kontrollkästchen aktivieren Sie die Anzeige von absoluten Werten auf der Y-Achse.

Prozentwerte

Mit diesem Kontrollkästchen aktivieren Sie die Anzeige von prozentualen Werten auf der Y-Achse.

Splits anzeigen

Mit diesem Kontrollkästchen können Sie ggf. Splits im Chart anzeigen lassen.

Kapitalmaßnahmen verrechnen

Mit diesem Kontrollkästchen wählen Sie, ob ggf. Kapitalmaßnahmen im Chart berücksichtigt werden sollen.

Dividenden verrechnen (hist. Kurse angepasst)

Mit diesem Kontrollkästchen wählen Sie, ob Dividenden in den dargestellten Chartverlauf berücksichtigt werden sollen.
