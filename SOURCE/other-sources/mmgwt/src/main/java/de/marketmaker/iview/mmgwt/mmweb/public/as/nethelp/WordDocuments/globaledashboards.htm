Globale Dashboards

Beim Starten der vwd advisory solution empfängt Sie das Programm mit dem globalen Dashboard, das Ihnen als Startseite zugewiesen ist.

Über das Icon "Dashboards" im Bereichsmenü wechseln Sie jederzeit zur Liste der verfügbaren (globalen) Dashboards. Im Navigator finden Sie diese in alphabetischer Reihenfolge – unterteilt in öffentliche und private Dashboards.

Hinweis:
Ein globales Dashboard hat kein Eingabeobjekt. Um ein globales Dashboard einem Depotobjekt (Inhaber oder Portfolio) zuzuweisen und damit als Inhaber- bzw. Portfolio-Dashboard im Objektfenster des Depotobjekts zu erscheinen, muss bei der Konfiguration der Vorlage der entsprechende Depotobjekt-Typ der Dashboard-Vorlage zugeordnet werden. Dabei können Sie entweder nur Inhaber, nur Portfolios oder auch beide Typen zuordnen.

Im vwd portfolio manager legen Sie in den Eigenschaften der Vorlagen vom Typ "Dashboard" auch fest, ob diese für Kundensitzungen verfügbar sein sollen.
