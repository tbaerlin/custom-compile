Objektansichten im Bereich "Märkte"

Wenn Sie ein Wertpapier z. B. per Mausklick auf den Namen des Wertpapiers in einer Kursliste auswählen, öffnet sich dessen Objektfenster. Sie sehen dann oben im Navigator den Steckbrief des Wertpapiers. Darunter finden Sie die Stammdaten aus dem vwd portfolio manager, ggf. verknüpfte Wertpapiere sowie verschiedene weitere Funktionen.

Sie gelangen in diese Objektansichten über die Suche oder durch Auswahl eines Wertpapiers in den Kurslisten usw. In den Listen und Übersichten im Auswertungsbereich wählen Sie ein Wertpapier wie folgt aus:

1.   Öffnen Sie z. B. über die Startseite "Übersicht" die Kursliste DAX Xetra.

2.   Klicken Sie auf die Bezeichnung der zu öffnenden Aktie (Spalte "Name").

Sie können ein Wertpapier aber auch aus einer Vermögensübersicht öffnen.

Das Wertpapier wird nun in seinem Objektfenster geöffnet.
