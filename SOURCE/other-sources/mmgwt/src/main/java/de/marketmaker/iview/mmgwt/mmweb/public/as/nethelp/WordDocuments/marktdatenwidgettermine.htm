Marktdatenwidget "Termine"  

Das Marktdatenwidget "Termine" zeigt die Termine eines Tages mit den Spalten "Unternehmen" und "Ereignis" an. Ereignisse sind z. B. Dividendenauszahlungen oder Hauptversammlungstermine.

Standardeinstellung ist der aktuelle Tag. Über den Kalender in der Symbolleiste des Widgets können Sie die Tagesvorschau für einen anderen Tag auswählen:

Über die Links in der Spalte "Unternehmen" wechseln Sie direkt aus dem Dashboard in das entsprechende Wertpapierporträt.
